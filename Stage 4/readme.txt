How to run the program: 

clone the repo
git clone https://github.com/adrian3092/CS234_BusTicketManagementSystem.git

in file explorer, navigate to "CS234_BusTicketManagementSystem/Stage 4/dist"

double click "Stage_4.jar" to run the program

note: if using a Mac, and you get the following error:
"Apple could not verify “Stage_4.jar” is free of malware that may harm your Mac or compromise your privacy"
please run xattr -d com.apple.quarantine "CS234_BusTicketManagementSystem/Stage 4/dist/Stage_4.jar" and try again
